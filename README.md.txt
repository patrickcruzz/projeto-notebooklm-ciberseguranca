# Projeto NotebookLM — Estudos em Cibersegurança

## Objetivo
Criar um caderno temático utilizando NotebookLM para organizar estudos em redes, Linux, Kali Linux e Ethical Hacking.

---

## Fontes Utilizadas

- PDF Redes Cisco
- PDF Linux
- PDF Kali Linux
- PDF Ethical Hacking

---

## Engenharia de Prompts

### Prompt 1
"Resuma os conceitos principais das redes Cisco"

### Prompt 2
"Quais ferramentas de segurança o Kali Linux oferece?"

### Prompt 3
"Quais as diferenças entre o Kali e o Debian?"

### Prompt 4
"Cite os principais passos para quem está iniciando na cibersegurança hoje"

### Prompt 5
"Quais laboratórios um estudante de ciber deve ter?"

## Aprendizados

Durante o desenvolvimento deste caderno temático, foi possível compreender conceitos relacionados a redes Cisco, Linux, Kali Linux e fundamentos de cibersegurança.

### Conceitos aprendidos

- Estrutura básica de redes
- Ferramentas de segurança presentes no Kali Linux
- Diferenças entre Kali Linux e Debian
- Primeiros passos para iniciar na área de cibersegurança
- Importância de laboratórios virtuais
